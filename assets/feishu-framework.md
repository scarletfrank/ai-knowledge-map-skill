# Feishu 知识库框架结构

## 根目录结构
AI 知识地图 (Wiki Space)
├── 核心概念 (Folder)
│   ├── [领域] 概念组 (Folder)
│   │   ├── 具体概念文档 (Docx)
│   │   └── ...
│   └── ...
├── 文章笔记 (Folder)  
│   ├── YYYY-MM-DD 文章标题 (Docx)
│   └── ...
└── 索引页面 (Index Doc)

## 文档模板

### 核心概念文档模板
```markdown
# [概念名称]

## 定义
[概念的精确定义]

## 关键特性
- 特性1
- 特性2
- ...

## 相关概念
- [[相关概念1]]
- [[相关概念2]]
- ...

## 应用场景
[具体应用场景说明]

## 参考资料
- [来源1]
- [来源2]
```

### 文章笔记模板
```markdown
# [文章标题] - YYYY-MM-DD

## 核心观点
[文章的主要观点摘要]

## 提取的概念
- [[概念1]]: [简要说明]
- [[概念2]]: [简要说明]

## 个人见解
[你的思考和见解]

## 相关链接
- 原文: [URL]
- 相关概念: [[概念1]], [[概念2]]
```

### 索引页面模板
```markdown
# AI 知识地图索引

## 核心概念
- [概念分类1](folder_link)
  - [[概念1]]
  - [[概念2]]
- [概念分类2](folder_link)
  - [[概念3]]
  - [[概念4]]

## 最新文章笔记
- [[2026-02-24 文章标题]]
- [[2026-02-23 文章标题]]

## 统计信息
- 总概念数: X
- 总文章数: Y
- 最后更新: YYYY-MM-DD
```

## 权限要求
- 用户需要提供一个可访问的飞书知识库（Wiki Space）
- 应用需要以下权限：
  - `wiki:wiki` - 知识库读写权限
  - `docx:document` - 文档读写权限
  - `drive:file` - 文件操作权限

## 初始化流程
1. 用户提供目标知识库链接或 token
2. 技能验证权限并创建根目录结构
3. 创建索引页面和基础文件夹
4. 配置 RSS 源和监控设置
